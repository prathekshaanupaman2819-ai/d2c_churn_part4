# d2c_churn_part4
#Part 4: Exploratory Analysis & Churn-Risk Hypotheses

​This repository contains the comprehensive exploratory analysis and hypothesis testing for churn mitigation, as required by the investigative framework.

​## Project Contents

​exploratory_analysis_report.md: The formal report detailing analysis across customer demographics, order behavior, monetary behavior, support tickets, return/refund activity, web/app engagement, campaign history, and churn distribution.
​churn_risk_hypotheses.md: The documentation of five distinct churn-risk hypotheses, each supported by visual evidence (charts/tables) and brief explanations.
​pre_campaign_memo.md: The business memo outlining necessary investigations the company should perform before launching any retention campaigns.

​## How to Review

​Review the exploratory_analysis_report.md to understand the foundational data trends and patterns.
​Consult the churn_risk_hypotheses.md for data-backed drivers of churn.
​Read the pre_campaign_memo.md for strategic recommendations and investigative priorities.

​## Academic Integrity

​Analysis is grounded in the provided datasets and supported by visual evidence in accordance with the project requirements.
